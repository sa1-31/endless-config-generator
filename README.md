# Endless Config Generator

## What
A lightweight, mobile-first config generator that helps developers bootstrap Endless projects quickly using a simple web interface.

## Why
New developers often struggle with initial configuration and network setup. This tool reduces friction and clarifies the Endless project structure.

## Benefit to Endless
- Faster onboarding for new developers
- Clear mental model for project setup
- Reusable utility for the ecosystem
